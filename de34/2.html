Arduinoの機能を組み合わせて作品作り

①作品タイトル：社会風刺


ディズニーの顔ミッキーとUSJのミニオンが戦うようにステッピングモーターを組み合わせて 明るい状態では静止し、暗い状態だと動くようにした。これにより、表では争わないが、 裏では実はバチバチなのかもしれないと思わせるような作品になっている。


②使用したもの

・Arduino
・ステッピングモーター
・モータードライバ
・明るさセンサー
・白画用紙


③仕組み

明るさセンサーの値（int thresholdの値）を下回ると暗いとみなされ、モーターは動き、上回ると 明るいとみなされ、モーターは止まる

④実行したプログラム

const int DIR  = 8;
const int STEP = 9;

int stepsPerRevolution = 200;
int threshold = 200;

void setup() {
  pinMode(DIR, OUTPUT);
  pinMode(STEP, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  int val = analogRead(1);
  Serial.println(val);

  if (val > threshold) {
    // 暗い → 往復ビンタ
    rotateAngle(90, HIGH); // 右
    delay(10);
    rotateAngle(90, LOW);  // 左
    delay(10);
  } else {
    // 明るい → 停止
    digitalWrite(STEP, LOW);
  }
}

void rotateAngle(float angle, int direction) {
  int steps = stepsPerRevolution * angle / 360;

  digitalWrite(DIR, direction);

  for (int i = 0; i < steps; i++) {
    digitalWrite(STEP, HIGH);
    delayMicroseconds(1000);
    digitalWrite(STEP, LOW);
    delayMicroseconds(delaytime);
  }
}
