flaskについて


・ flaskとはPythonのwebアプリケーションフレームワーク。
・小規模向けの簡単なwebアプリケーションを作るのに適している。
webフレームワークとは、ウェブサイトやウェブアプリケーションを作るための 機能を提供し、ウェブフレームワークを使わない時よりもより容易にwebアプリケーション を作ることができるもの。

flaskのプログラム例

① Flaskの基本


from flask import Flask

app = Flask(__name__)

@app.route('/')
def index():
    return "Hello Flask!"

このコードはFlaskの基本構造で、トップページにアクセスすると文字を表示する。

② テンプレートの使用


from flask import render_template

@app.route('/')
def index():
    return render_template('index.html')

HTMLテンプレートを使うことで、より柔軟なページ表示が可能になる。

プログラムの説明

app = Flask(__name__)：Flaskアプリケーションの本体を作成するための記述である。


app.route：URLと処理を対応付けるための機能である。 指定したURLにアクセスされたときに、対応する関数が実行される。


def index()：指定されたURLにアクセスされたときに実行される関数である。


return "Hello Flask!"：ブラウザに表示する内容を返す記述である。 （今回のコマンドはアクセスされたときに「Hellp Flask!」と表示される）


render_template：FlaskでHTMLテンプレートを表示するための関数である。 templatesフォルダ内のHTMLファイルを読み込み、ブラウザに表示する役割を持つ。 これにより、Pythonの処理とHTMLのデザインを分離することができる。
